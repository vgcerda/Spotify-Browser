# Spotify Browser

Usage:
	Prerequisite:
		Have Spotify Account
		Create an application in the Spotify Developers Dashboard
			Set url to localhost:4200 and url callback to localhost:8888/callback
		Edit webserver/client_secret.json to include the correct client id and client secret from the Spotify Dashboard application settings
		
	Client:
		cd into the client folder and type 'ng serve'

	Webserver:
		cd into the webserver folder and type 'npm start'

	Web Browser:
		visit http://localhost:4200/ and browse features
